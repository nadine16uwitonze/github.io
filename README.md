## 🚀 How to Run

No installation needed. Just:

1. Download `nadine_uwitonze_portfolio.html`
2. Double-click it to open in any browser (Chrome, Firefox, Edge, Safari)

That's it — it works completely offline.

## 🌐 How to Host Online (Free)

### Option 1 — GitHub Pages
1. Create a new GitHub repository
2. Upload `nadine_uwitonze_portfolio.html` and rename it to `index.html`
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch and click **Save**
5. Your portfolio will be live at `https://yourusername.github.io/repository-name`

### Option 2 — Netlify Drop
1. Go to [netlify.com/drop](https://netlify.com/drop)
2. Drag and drop your `nadine_uwitonze_portfolio.html` file
3. Your site is instantly live with a public URL
